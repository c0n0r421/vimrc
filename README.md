Plugins

NERDTree:

Space + e → toggle file explorer

FZF:

Space + f → fuzzy file search

Commentary:

Space + / → toggle comment on line or selection

Surround:

cs"' → change surrounding " to '

ysiw] → wrap word in []

ds" → delete surrounding "

Fugitive (Git):

:Git status → view status

:Git add % → stage current file

:Git commit → commit

:Git push → push

Space + gb → blame current file

ALE (Linting/Format):

Auto-formats Python with Black and isort on save

Space + n → next linting error

Space + p → previous linting error
